# Evaluation Framework

Sistema de evaluación para el proyecto de tesis de autenticación biométrica por voz.

## Estructura

```
evaluation/
├── scripts/              # Scripts de evaluación automatizada
├── dataset/             # Datasets de prueba
├── results/             # Resultados de evaluaciones
├── metrics_calculator.py # Cálculo de métricas biométricas
└── evaluation_logger.py  # Logging para evaluación manual
```

## Uso

Ver `evaluation_guide.md` para instrucciones detalladas.
